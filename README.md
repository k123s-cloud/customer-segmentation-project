Customer Segmentation Project 🚀

This project uses Machine Learning to classify customers into different segments like Loyal, Potential, etc.

🔥 Features

- End-to-end ML pipeline
- FastAPI backend
- Customer prediction UI
- Model training & evaluation

🛠️ Tech Stack

- Python
- FastAPI
- Scikit-learn
- Pandas

▶️ Run Locally

pip install -r requirements.txt
python app.py

📊 Output

Customer is classified into clusters such as:

- Loyal Customer
- Potential Customer
- At Risk Customer
